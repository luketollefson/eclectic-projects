Globosus Web Application
========================

## Prerequasists

You obviously need git intalled. Install it using your package manager.

Ensure you have stack installed. View [this](https://docs.haskellstack.org/en/stable/README/) to learn how to install stack.

## Getting Globosus

Run the following:

`git clone https://github.com/luketollefson/globosus-web.git`

## Running Globosus

Go into the globosus-web directory and execute this

`stack exec hello`

Then navigate using your browser to `http://localhost:8000/` and walla! you have Globosus
